# Oracle DBA Toolkit

A curated collection of Oracle DBA scripts for performance tuning, monitoring, backup, and administration.

## Structure

| Directory | Source | Description |
|---|---|---|
| `fatdba-scripts/` | [fatdba](https://github.com/Nirmal7086/Oracle-Database-Scripts_fatdba) | Performance tuning, ASH/AWR, RAC, Data Guard, RMAN, and general admin scripts |
| `luca-canali-scripts/` | [Luca Canali](https://github.com/Nirmal7086/Oracle_DBA_scripts_LucaCanali) | ASH, AWR reports, session monitoring, locks, and SQL diagnostics |
| `robkoltai-scripts/` | [Rob Koltai](https://github.com/Nirmal7086/robkoltai_DBA_scripts) | SQL tuning, CBO stats, RAC, RMAN, Data Guard, SPM, and OS perf collection |
| `windwiny-tips-scripts/` | [windwiny](https://github.com/Nirmal7086/windwiny_Oracle-DBA-tips-scripts) | SQL*Plus utilities, external procedures, and helper scripts |
| `tpt-oracle-scripts/` | [Tanel Poder](https://github.com/Nirmal7086/tpt-oracle) | Extensive SQL*Plus toolkit for deep Oracle internals, diagnostics, and tracing |
